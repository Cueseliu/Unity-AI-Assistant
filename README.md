# AI Operator - Your Unity AI Assistant

**Talk to Unity. Get things done.**

AI Operator is a revolutionary Unity Editor plugin that transforms how you work with Unity. Simply describe what you want in plain English (or Chinese), and watch as the AI executes your commands directly in the editor.

## Key Features

### Natural Language Control
- Create GameObjects with a simple "Create a red Cube"
- Modify transforms: "Move Player to (0, 5, 0)"
- Batch operations: "Create 10 enemies in a circle"
- No scripting required for common tasks

### 30+ Built-in Tools
- **GameObject**: Create, delete, duplicate, find, transform
- **Components**: Add, remove, query any Unity component
- **Materials**: Set colors and properties
- **UI**: Create Canvas, Button, Text, Slider, Panel, Image
- **Prefabs**: Save and instantiate prefabs
- **Scripts**: Generate C# scripts automatically
- **Scene**: Analyze and manage scene hierarchy

### Multiple AI Services
Choose your preferred AI provider:
- **Claude** (Anthropic) - Best performance, recommended
- **DeepSeek** - Affordable, great value
- **AWS Bedrock** - Enterprise-grade, Claude via AWS
- **Qwen** - Alibaba Cloud service
- **GLM-4** - Zhipu AI
- **Doubao** - ByteDance service

### Workflow Automation
One command does it all:
- "Create a player character with WASD movement"
  → Creates GameObject + Rigidbody + Collider + Script + Prefab
- "Create a main menu with start and exit buttons"
  → Creates Canvas + EventSystem + Buttons + Layout

### Bilingual Interface
- Full Chinese and English UI support
- Switch languages anytime
- Documentation in both languages

### Developer-Friendly
- Config import/export for team sharing
- Command history for quick re-use
- Keyboard shortcuts for efficiency
- Detailed tool execution logs

## Use Cases

- **Rapid Prototyping**: Quickly create scenes and test ideas
- **Learning Unity**: Natural way to explore Unity features
- **Productivity Boost**: Automate repetitive tasks
- **Team Collaboration**: Share configurations easily

## Requirements

- Unity 2021.3 or higher
- Internet connection
- API Key from supported AI service

## Getting Started

1. Import the package
2. Open Window > AI Operator > Settings
3. Enter your API Key
4. Start chatting!

# AI Operator - Unity AI 智能助手

**和 Unity 对话，让事情自动完成。**

AI Operator 是一个革命性的 Unity 编辑器插件，彻底改变你使用 Unity 的方式。只需用自然语言描述你想做什么，AI 就会直接在编辑器中执行你的命令。

## 核心功能

### 自然语言控制
- 一句话创建物体："创建一个红色的 Cube"
- 修改 Transform："把 Player 移动到 (0, 5, 0)"
- 批量操作："创建 10 个敌人排成一圈"
- 常见操作无需编写代码

### 30+ 内置工具
- **GameObject**: 创建、删除、复制、查找、变换
- **组件**: 添加、移除、查询任意 Unity 组件
- **材质**: 设置颜色和属性
- **UI**: 创建 Canvas、Button、Text、Slider、Panel、Image
- **Prefab**: 保存和实例化预制体
- **脚本**: 自动生成 C# 脚本
- **场景**: 分析和管理场景层级

### 多 AI 服务支持
选择你喜欢的 AI 服务：
- **Claude** (Anthropic) - 效果最好，推荐使用
- **DeepSeek** - 价格实惠，性价比高

- **通义千问** - 阿里云服务
- **智谱 GLM-4** - 国产大模型
- **豆包** - 字节跳动服务

### 工作流自动化
一句话完成复杂流程：
- "创建一个玩家角色，可以 WASD 移动"
  → 自动创建 GameObject + Rigidbody + Collider + 脚本 + Prefab
- "创建一个主菜单，包含开始和退出按钮"
  → 自动创建 Canvas + EventSystem + 按钮 + 布局
